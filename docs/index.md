# MixedBerryPie
{: .fs-9 }
指先に宿るショートカット。高速・高機能な Windows 用パイメニュー。
{: .fs-6 .fw-300 }

**MixedBerryPie** は、グローバルホットキーを押すだけでマウスポインターの位置にパイメニュー（円形メニュー）を展開する Windows 用ショートカットランチャーです。  
ペンタブレットでイラストを描きながら、キーボードに手を伸ばすことなく片手でショートカットを呼び出すといった用途を想定して設計されています。

## アプリケーションの特長

- **究極のゼロレイテンシー**  
  Windows の低レベルネイティブフック（`WH_KEYBOARD_LL`）を使用し、OSレベルでキーボードイベントを捕捉。起動の遅延が一切なく、他のアプリへの誤入力（キーの暴発）を完全に防ぎます。

- **高度なコンテキスト認識**  
  「現在アクティブなウィンドウ（アプリ名やタイトル）」を自動で判別。Photoshopを使っている時はPhotoshop用のメニュー、ブラウザを使っている時はブラウザ用のメニューへと、瞬時にプロファイルが切り替わります。

- **多彩で安全なアクション**  
  単なるキー送信だけでなく、URLの展開、コマンドプロンプトの実行、さらに入れ子になる「サブメニュー」にも対応。また、キーストローク送信前には修飾キー（CtrlやShiftなど）の押しっぱなし状態を安全に解除（Release）するため、複雑なショートカットも確実に入力されます。

- **自由自在なカスタマイズUI**  
  設定画面で色（美しいカラープリセット搭載）、アイコン、透明度、メニューの描画サイズ、動作ディレイなどをリアルタイムプレビューを見ながら直感的に設定できます。

- **徹底したプライバシー保護**  
  すべてのショートカット・マクロ処理はPCのローカル上で完結します。打鍵情報の収集や外部サーバーへの送信は一切行いません。

## インストール方法

### Microsoft Store (自動更新・推奨)

ストア版は常に最新バージョンへ自動的にアップデートされます。

[![Microsoft Store から入手](https://get.microsoft.com/images/ja-jp%20dark.svg)](https://apps.microsoft.com/detail/9ns29508pnzp?hl=ja-JP&gl=JP)

### GitHub Releases (手動インストール)

1. [Releases ページ](https://github.com/Renge2212/MixedBerryPie/releases) から、最新の `MixedBerryPie-*-Setup.exe` をダウンロードします。
2. インストーラーを実行してPCにインストールします。
3. アプリを起動すると、タスクバー右下の「システムトレイ」にアプリのアイコンが常駐します。
