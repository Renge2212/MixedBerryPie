## プロファイルとコンテキスト切り替え

プロファイルとは、メニュー項目（スライス）の集合のことです。
MixedBerryPie は、**現在アクティブなウィンドウ（対象アプリ）に応じてプロファイルを自動的に切り替える**「コンテキスト認識」機能を備えています。

### アプリ連動の設定方法

1. 設定画面左側のリストの下にある「**＋ 追加**」ボタンをクリックして新しいプロファイルを作ります。
2. そのプロファイルを選択した状態で、画面右側の「**対象アプリ**」欄をクリックします。
3. **実行ファイル名**（例: `photoshop.exe` や `chrome.exe`）を入力してエンターを押します。
4. 対象アプリは複数登録することもできます。

![設定画面で対象アプリ（コンテキスト）を設定している様子](../assets/images/context_settings.png)

> **💡 ヒント**: 「対象アプリ」が何も設定されていないプロファイルは **デフォルトプロファイル (Default)** として扱われ、他のどのアプリルールにも一致しない場合に使用されます。

## アクション（スライス）の種類

各スライスには以下のいずれかのアクションを設定できます。
スライスは**最大12個まで**追加可能です。

| 種類 | 説明 | 設定例 |
|---|---|---|
| **キー送信** | 指定したショートカットキーを、OSのネイティブ制御で安全に現在アクティブなウィンドウへ送信します。 | `ctrl+z`, `f5`, `shift+alt+s` |
| **URL を開く** | デフォルトのウェブブラウザで指定URLを開きます。 | `https://example.com` |
| **コマンド実行** | バックグラウンドでシェルコマンドを実行します。 | `notepad.exe`, `explorer .` |
| **サブメニュー** | *（後述）*スライスを選択した際に、入れ子のメニューをさらに展開します。 | — |

### キー送信の書式 (修飾キー)

修飾キーと通常キーを `+`（プラス）で繋ぎます。大文字・小文字は区別しません。
OSネイティブレベルでの送信を行うため、**トリガーで押した修飾キーが悪さをして入力が意図せぬ動作になることを防ぎます。**（例：トリガーが `Shift` で、送信するキーが `a` のとき、単なる `A` になるのを防止します）

```
ctrl+c          # コピー
ctrl+shift+z    # やり直し
alt+f4          # ウィンドウを閉じる
f11             # フルスクリーン
```

## サブメニュー機能

一つのパイメニューだけでは項目が足りない場合、スライスのアクションを「**サブメニュー**」に設定することで、別のパイメニューを入れ子にすることができます。

### サブメニューの開き方
親メニューを表示させ、サブメニューが設定されたスライスにカーソルを合わせると、選択待ち状態になり子メニューが展開されます。
**そのままホットキーを離さずに**子メニューのスライスへカードを動かし、目的の場所でキーを離すとアクションが実行されます。

![サブメニューが展開されている様子](../assets/images/submenu_expanded.png)

### 特殊なレイアウト設定
サブメニューのデザインは、親メニューとの関係性によって2つのパターン（レイアウト）から選べます。

- **同心円 (Concentric) レイアウト**:
  親メニューの外側に、新しい円（リング）としてメニューが広がります。画面の中央を基準に大きく展開したい場合に向けたデザインです。
- **扇形 (Fan) レイアウト**:
  選択した親スライスの方向に、まるで扇子が開くようにメニューが展開されます。マウスの移動距離を短く抑えたい直感的な操作向けです。

## 外観（テーマ表示）の設定

設定画面の **「外観 (Appearance)」タブ** では、メニューの見た目を大きく変更できます。
変更はすべて右側のプレビュー画面にリアルタイムで反映されます。

![外観設定タブとリアルタイムプレビューの様子](../assets/images/appearance_settings.png)

### カラーモードの設定
MixedBerryPie は以下の3種類のカラーモードを持っています。

1. **プリセット (Preset)**: "Mixed Berry", "Cyberpunk", "Ocean" などの、美しく調整された独自のカラーテーマを一括で適用します。
2. **単一カラー (Unified)**: 全スライスを指定した1色で統一します（ミニマルなデザインを好む方向け）。
3. **個別カラー (Individual)**: スライス作成時に自分で設定・微調整した固有の色をそのまま使用します。

### 表示サイズと透明度
- **メニューサイズ**: パイメニュー全体の直径（ピクセル）
- **背景色・透明度 (Opacity)**: メニュー背景のカラーと不透明度
- **ハイライト色**: 選択中スライスの強調カラー
- **テキストサイズ & アイコンサイズ**: スライス内のラベルの表示サイズ。
  > デフォルトでは **「メニューサイズに合わせて自動調整 (Auto scale)」** がオンになっていますが、オフにすることで自由にピクセルで指定可能になります。
